## 1.0.1 （2020-12-10）

### Bug Fixes

- **Button:** 支持 style 扩展
- **Icon：** 新增图标及规范图标命名
- **Checkbox:** group 组件支持透传扩展样式
- **Textarea:** 优化 textarea 选中高度
- **Select：** 修复 select 组件销毁时的错误
- **Tabs：** 组件样式优化
- **Tag:** 调整样式和 icon 大小和默认色值
- **Alert：** 修复控制台警告
- **Modal:** UI 升级&内部逻辑优化
- **Table:** 增加默认分页器，修复缺省数据高度展示

### Features

- **Input:** 支持 searchInput 功能
- **From:** 增加开放校验范围
- 升级 react/react-dom 版本（17.0.2）
- lerna 安装依赖配置修改
