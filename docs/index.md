## 轻松上手

### taro 千牛组件库

```bash
// 安装依赖
npm i taro-qn-components -S

// 使用组件
import { CButton } from 'taro-qn-components';

```

### cloud-react 组件库

```bash
npm i cloud-react -S

import { Button } from 'cloud-react';
import 'cloud-react/cloud-react.css';

ReactDOM.render(<Button>按钮</Button>, mountNode);

```
